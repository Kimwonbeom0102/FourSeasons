# FourSeasons

**프로젝션 매핑을 활용한 미디어 작품**

Blender, Unity, MadMapper를 이용해 사계절을 주제로 제작된 미디어 아트입니다.  
프로젝션 매핑을 통해 실제 공간에 시각적 몰입을 연출했습니다.

---

## 🔧 사용 툴  
- **Blender**: 3D 모델링 및 애니메이션  
- **Unity**: 씬 구성 및 시각효과  
- **MadMapper**: 프로젝션 매핑 출력
